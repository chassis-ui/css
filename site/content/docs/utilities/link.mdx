## Link colors

`.link-*` classes apply context colors to links and include interactive state styles (`:hover`, `:focus`, `:active`, `:visited`) — unlike `.fg-*` utilities, which are static. Some link color variants use light foreground values and are intended for use on dark backgrounds only.

<Example code={getData('context-colors').map((color) => `<p><a href="#" class="link-${color.name}">${titleCase(color.name)} link</a> — <a href="https://example.com" class="link-${color.name}">visited link</a></p>`)} />

<Callout name="warning-color-assistive-technologies" />

### Reset color

`.fg-reset` removes the foreground color override, causing the element to inherit color from its nearest ancestor.

<Example code={`<p class="fg-subtle">
    Secondary body text with a <a href="#" class="fg-reset">reset link</a>.
  </p>
  <p class="fg-subtle">
    Secondary body text with a <a href="#">normal link</a>.
  </p>`} />

## Link opacity

Link opacity utilities adjust the alpha channel of link color values. Reduced opacity can impact color contrast — review contrast levels when using these utilities.

<Example code={[...getData('token-opacities').map((opacity) => `<p><a href="#" class="link-opacity-${opacity.name}">Link opacity ${opacity.name} link</a></p>`)]} />
<Example code={[...getData('token-opacities').map((opacity) => `<p><a href="#" class="link-secondary link-opacity-${opacity.name}">Link opacity ${opacity.name} link</a></p>`)]} />

### Hover variants

Hover-specific opacity variants apply the opacity value only on `:hover`.

<Example code={[...getData('token-opacities').map((opacity) => `<p><a href="#" class="hover:link-opacity-${opacity.name}">Link hover opacity ${opacity.name} link</a></p>`)]} />

## Link underlines

Underline utilities modify the visual presentation of link underlines independently from link text color.

### Underline color

The underline color can be set independently from the link text color.

<Example code={getData('context-colors').map((contextColor) => `<p><a href="#" class="text-decoration-${contextColor.name}">${titleCase(contextColor.title)} underline</a></p>`)} />

### Underline offset

Underline offset utilities set `text-underline-offset` in `em` units, scaling automatically with the element’s font size.

<Example code={`<p><a href="#">Default link</a></p>
<p><a class="underline-offset-1" href="#">Offset 1 link</a></p>
<p><a class="underline-offset-2" href="#">Offset 2 link</a></p>
<p><a class="underline-offset-3" href="#">Offset 3 link</a></p>`} />

### Underline opacity

`.text-decoration-opacity-{value}` adjusts the alpha channel of the decoration color. Apply it alongside `.text-decoration` or any `.text-decoration-{color}` class, which set the decoration color as an opacity-aware value.

<Example code={[...getData('token-opacities').map((opacity) => `<p><a href="#" class="underline-offset-2 text-decoration-info text-decoration-opacity-${opacity.name}">Underline opacity ${opacity.name}</a></p>`)]} />

### Hover variants

Hover-specific decoration opacity variants apply the opacity value only on `:hover`.

<Example code={`<a class="text-decoration text-decoration-opacity-zero hover:text-decoration-opacity-50 underline-offset-3" href="#">
    Underline opacity 0
  </a>`} />
