## Overview

Float utilities set the CSS [`float` property](https://developer.mozilla.org/en-US/docs/Web/CSS/float) — `start`, `end`, or `none` — based on the viewport size. `!important` is applied to prevent specificity conflicts. These utilities follow the same breakpoint scale as the grid. Note: float has no effect inside flex containers.

<Example code={`<div class="float-start">Float start on all viewport sizes</div><br>
<div class="float-end">Float end on all viewport sizes</div><br>
<div class="float-none">Don’t float on all viewport sizes</div>`} />

Use the [clearfix utility]([[docsref:/helpers/clearfix]]) on a parent element to clear floats.

## Responsive

Responsive variants are available for all float values:

<Example code={`<div class="small:float-end">Float end on viewports sized "<b>small</b>" or wider</div><br>
<div class="medium:float-end">Float end on viewports sized "<b>medium</b>" or wider</div><br>
<div class="large:float-end">Float end on viewports sized "<b>large</b>" or wider</div><br>
<div class="xlarge:float-end">Float end on viewports sized "<b>xlarge</b>" or wider</div><br>
<div class="2xlarge:float-end">Float end on viewports sized "<b>2xlarge</b>" or wider</div><br>`} />

Here are all the support classes:
