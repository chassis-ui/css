The `.opacity-{value}` utilities set the CSS `opacity` property on an element. A value of `1` is fully opaque, `.5` is 50% visible, and `0` is fully transparent.

<div class="cxd-example d-flex flex-wrap">
  {
    getData('token-opacities').map((opacity) => {
      return <div class={`bg-opacity-${opacity.name} fg-${opacity.name === 'solid' || opacity.name > 50 ? 'alternate' : 'default' } bg-default fg-opacity-80 p-small m-xsmall rounded`}>{opacity.name}</div>
    })
  }
</div>

## Per-property opacity

Several utility types — background, foreground, and border — support per-property opacity through CSS variable overrides, allowing opacity changes without affecting the entire element.

### How it works

These utilities depend on CSS relative color syntax. The color utility sets both the color value and a `--cx-{type}-opacity` variable (defaulting to `1`). A paired opacity utility overrides that variable to adjust transparency.

```css
.bg-primary {
  background-color: oklch(from var(--cx-primary) l c h / var(--cx-bg-opacity, 1)) !important;
}
.bg-opacity-50 {
  --cx-bg-opacity: var(--cx-opacity-50);
}
```

### Opacity override

The opacity variable can be set inline or via a `.bg-opacity-*` utility.

<Example code={`<div class="bg-primary p-xsmall primary-contrast-color">
    Primary background
  </div>
  <div style="--cx-bg-opacity: .4;" class="bg-primary p-xsmall">
    Primary background with 40% opacity
  </div>`} />

## Context opacities

Context-named opacity utilities correspond to the design token scale and can be applied to foreground, background, and border properties.

### Foreground

Foreground opacity utilities apply context-named opacity values to `.fg-*` color classes.

<Example code={`<p class="fg-primary">Base color</p>
<p class="fg-primary fg-opacity-subtle"><code>fg-subtle</code> opacity</p>
<p class="fg-primary fg-opacity-slight"><code>fg-slight</code> opacity</p>`} />

Check out [color]([[docsref:/utilities/colors#opacity]]) page for more details.

### Background

Background opacity utilities apply scene-token opacity values to `.bg-*` color classes.

<Example code={`<div class="bg-primary p-xsmall">Base background</div>
<div class="bg-primary bg-opacity-main p-xsmall"><code>dim-main</code> opacity</div>
<div class="bg-primary bg-opacity-subtle p-xsmall"><code>dim-subtle</code> opacity</div>
<div class="bg-primary bg-opacity-slight p-xsmall"><code>dim-slight</code> opacity</div>`} />

Check out [background]([[docsref:/utilities/background#opacity]]) page for more details.

### Border

Border opacity utilities apply context-named opacity values to `.border-*` color classes.

<Example code={`<p class="border border-primary p-xsmall">Base border</p>
<p class="border border-primary border-opacity-main p-xsmall"><code>border-main</code> opacity</p>
<p class="border border-primary border-opacity-subtle p-xsmall"><code>border-subtle</code> opacity</p>`} />

Check out [borders]([[docsref:/utilities/borders#opacity]]) page for more details.
