## Overview

Visibility utilities set the CSS `visibility` property. Unlike display utilities, they do not affect layout — hidden elements retain their space in the document flow.

<Callout type="warning">
`.invisible` elements are hidden both visually and from assistive technologies.
</Callout>

`.visible` and `.invisible` are the available classes.

```html
<div class="visible">...</div>
<div class="invisible">...</div>
```

```scss
// Class
.visible {
  visibility: visible !important;
}
.invisible {
  visibility: hidden !important;
}
```
